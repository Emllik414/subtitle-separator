双语字幕工具（Windows 免安装版）
================================

使用方法

1. 先把整个 ZIP 解压到一个固定文件夹。
2. 直接双击 SubtitleSeparator.exe 即可使用，不需要安装。
3. 需要桌面快捷方式时，双击 Create Desktop Shortcut.cmd。
4. 创建成功后，桌面会出现“双语字幕工具”快捷方式。

注意事项

- 不要只把 EXE 单独拖出压缩包运行，建议先完整解压。
- 移动软件所在文件夹后，请重新运行 Create Desktop Shortcut.cmd。
- Windows 首次运行可能显示 SmartScreen 提示，这是未购买代码签名证书导致的。点击“更多信息”后选择“仍要运行”。
- 软件不会写入系统安装目录，也不需要管理员权限。
